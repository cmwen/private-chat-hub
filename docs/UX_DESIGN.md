**Document Version:** 1.0  
**Created:** December 31, 2025  
**Status:** Design Complete  
**Author:** @experience-designer Agent

## Design Philosophy

### Core Principles

1. **Familiar Yet Powerful**: Feel like ChatGPT/Claude but with local-first superpowers
2. **Transparency**: Always show connection status, model info, and what's happening
3. **Speed**: Minimize taps to common actions (send message, switch model)
4. **Privacy-First**: Visual cues that reinforce data stays local
5. **Progressive Disclosure**: Simple by default, powerful when needed

### Design Language

- **Material Design 3** (Material You)
- **Dynamic Color**: Adapt to user's wallpaper/theme
- **Large touch targets**: 48dp minimum
- **Readable typography**: Support system font scaling
- **Dark mode first**: Privacy-conscious users often prefer dark themes

## 5. Accessibility

### 5.1 Screen Reader Support

| Element | Semantic Label |
|---------|----------------|
| Connection indicator | "Connected to Home Server" / "Disconnected" |
| Model selector | "Current model: llama3.2. Tap to change" |
| Send button | "Send message" |
| Attachment button | "Add attachment" |
| User message | "You said: [content]" |
| AI message | "[Model name] replied: [content]" |
| Copy button | "Copy message to clipboard" |
| Code block | "Code block: [language]. Tap to copy" |

### 5.2 Touch Targets

- Minimum touch target: **48dp Ã— 48dp**
- Adequate spacing between interactive elements
- Swipe actions have visual hints

### 5.3 Color Contrast

- Text on background: **4.5:1** minimum (WCAG AA)
- Large text: **3:1** minimum
- Interactive elements clearly distinguishable
- Don't rely solely on color (use icons + color)

### 5.4 Text Scaling

- Support system font size up to 200%
- Layout adapts without truncation
- Multi-line text fields expand appropriately

| Transition | Animation | Duration |
|------------|-----------|----------|

| Drawer open/close | Slide from left | 250ms |
| Bottom sheet | Slide from bottom | 250ms |
| Modal dialog | Fade + scale | 200ms |
| Message appear | Fade in + slide up | 150ms |
| Send message | Slide right | 200ms |

| Interaction | Feedback |
|-------------|----------|
| Tap send button | Ripple + haptic |
| Long press message | Haptic + context menu |
| Swipe conversation | Reveal action + haptic at threshold |
| Connection restored | Snackbar + subtle animation |
| Download complete | Notification + celebration |

```

- Connected: #4CAF50 (Green)
- Connecting: #FFC107 (Amber)
- Disconnected: #F44336 (Red)

Hardware Compatibility:
- Recommended: #4CAF50 (Green)
- Moderate: #FFC107 (Amber)
- Heavy: #F44336 (Red)

### 8.2 Typography

```
Headlines: Roboto Medium
Body: Roboto Regular
Code: JetBrains Mono / Fira Code

Sizes (scaled with system):
- Title Large: 22sp
- Title Medium: 16sp
- Body Large: 16sp
- Body Medium: 14sp
- Label: 12sp
```

### 8.3 Spacing

```
Base unit: 4dp

Spacing scale:
- xs: 4dp
- sm: 8dp
- md: 16dp
- lg: 24dp
- xl: 32dp

Content padding: 16dp
Card padding: 16dp
