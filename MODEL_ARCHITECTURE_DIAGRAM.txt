  ModelCapabilityResolver.getCapabilities('qwen2.5:7b')

         ▼
  ModelRegistry.getCapabilities('qwen2.5:7b')
         ▼
  Normalize: 'qwen2.5' (removes ':7b')
         ▼
  Direct lookup in registry...
         ▼
  FOUND! Returns ModelCapabilities(
    supportsToolCalling: true,
    supportsVision: false,
    contextWindow: 128000,
    modelFamily: 'qwen',
    ...
  )

         ▼
  Starts with 'local:' ? YES
         ▼
  OnDeviceModelCapabilitiesRegistry.getCapabilities('local:gemma3-1b')
         ▼
  Remove 'local:' prefix, normalize: 'gemma3-1b'
         ▼
  Direct lookup in registry...
         ▼
  FOUND! Returns ModelCapabilities(
    supportsToolCalling: true,
    supportsVision: false,
    contextWindow: 4096,
    modelFamily: 'gemma',
    ...
  )

  )

║                  CAPABILITY-BASED UI ADJUSTMENT                           ║
╚═══════════════════════════════════════════════════════════════════════════╝

Based on ModelCapabilities, Chat UI shows/hides:

supportsVision = true
  ▶ Show "Attach Image" button
  ▶ Enable image picker

supportsAudio = true
  ▶ Show "Attach Audio" button
  ▶ Enable audio recorder

supportsToolCalling = true
  ▶ Show "Enable Tools" toggle
  ▶ Show available tools list
  ▶ Enable tool execution

supportsThinking = true
  ▶ Show "Thinking Mode" toggle
  ▶ Display reasoning tokens

contextWindow value
  ▶ Show context usage percentage
  ▶ Warn when approaching limit

║                    ADDING NEW PROVIDERS (PATTERN)                         ║
╚═══════════════════════════════════════════════════════════════════════════╝

To add provider like OpenAI, Anthropic, or LM Studio:

1. CREATE MODEL REGISTRY
   File: lib/providers/[provider]_models.dart
   Class: [Provider]ModelRegistry
   
2. CREATE LLM SERVICE
   File: lib/services/[provider]_llm_service.dart
   Class: [Provider]LLMService implements LLMService
   
3. CREATE API CLIENT
   File: lib/[provider]/services/[provider]_client.dart
   Class: [Provider]Client
   
4. UPDATE MODEL RESOLVER
   File: lib/models/model_capability_resolver.dart
   Add: if (_is[Provider]Model(modelName)) { ... }
   
5. UPDATE CHAT SERVICE
   File: lib/services/chat_service.dart
   Add: if (model.startsWith('[provider]:')) { ... }


EXAMPLE: OpenAI Provider Setup
──────────────────────────────

'openai:gpt-4' ──┐
                 ├──> ModelCapabilityResolver ──> OpenAIModelRegistry
'openai:o1'  ───┤
                 │
'ollama:llama' ─┼──> ModelCapabilityResolver ──> ModelRegistry (Ollama)
                 │
'local:gemma' ──┘──> ModelCapabilityResolver ──> OnDeviceCapRegistry
